Dərsə başla
Müasir IT həllərdə ən geniş tətbiq edilən proqramlaşdırma dillərindən biri də Python proqramlaşdırma dilidir. Python dili sadə sintaksisi, daha çox sahələrə tətbiq edilmə imkanı, zəngin kitabxana seçimi, böyük developer icması və s. kimi amillərə görə digər proqramlaşdırma dillərindən fərqlənir.

Təqdim etdiyimiz "Pythonun əsasları" kursu proqramlaşdırmaya yeni başlayan istənilən şəxs üçün uyğundur. Bununla belə İnformasiya Texnologiyaları sahəsində çalışmaq istəyənlər, müəllim və tələbələr, data və biznes analitikası ilə məşğul olanlar və s. insanlar üçün də bu kurs tövsiyə edilir. Kursdakı dərslər yalnız nəzəri biliyi olan mütəxəssislər tərəfindən deyil, məhz öz işində Python dilini praktiki tətbiq edən mütəxəssislər tərəfindən hazırlanıb. Ona görə də bu onlayn dərslər sənə öyrəndiklərini praktiki tətbiq etmək üçün də imkanlar yaradacaq. Belə ki, hər mövzunun sonunda mövzuya aid çalışmalar və test tapşırıqları etməklə biliklərini möhkəmləndirəcəksən. Çox uzatmadan keçək dərslərə.
